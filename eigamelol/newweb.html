A new WEB??

Yes! I recently built a new version of my website and spent time improving it to make it more complete and polished. I fixed various bugs, refined unfinished sections, and made several adjustments to improve the overall user experience.

So I use SeaMonkey Composer HTML WYSIWYG to create and organize the HTML content efficiently, while Phoenix Code gives me complete control over the design and functionality when fine-tuning HTML, CSS, and JavaScript.

Instead of using a traditional database, I store website data in JSON files. Since my website is a static site, this approach keeps the project simple, lightweight, easy to maintain, and fast to load!

Finally, the complete project folder containing HTML, CSS, JavaScript, and JSON files is deployed to GitHub Pages and Vercel, making the site accessible to anyone on the web.
